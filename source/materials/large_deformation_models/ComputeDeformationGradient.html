ComputeDeformationGradient

This class computes the deformation gradient. Eigen deformation gradients are extracted from the total deformation gradient. The F-bar approach can optionally be used to correct volumetric locking.

Example Input File Syntax

    Options:

    Description:The displacements appropriate for the simulation geometry and coordinate system
