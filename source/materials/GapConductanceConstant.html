GapConductanceConstant

Description The GapConductanceConstant material provides a user-specified, constant gap condance that is used in thermal contact enforcement. This material model is typically not directly specified by the user, but is created using the ThermalContactAction, the details of which are documented in the ThermalContact system documentation page.

    C++ Type:std::vector<VariableName>

    Options:

    Description:Temperature variable
