Compute Finite Strain Elastic Stress

Compute stress using elasticity for finite strains

Description

This material, ComputeFiniteStrainElasticStress computes the elastic stress for an incremental formulation, both incremental small (ComputeIncrementalSmallStrain type) and incremental finite (ComputeFiniteStrain type) strain formulations. This stress class is compatible with both Cartesian and non-Cartesian strain calculators, including Axisymmetric and RSpherical.

Elastic materials do not experience permanent deformation, and all elastic strain and elastic stress is recoverable. Elastic stress is related to elastic strain through the elasticity tensor (1) where is the strain increment; this strain measure is also the sum of the mechanical elastic strain and any eigenstrains in the system.

Example Input File Syntax

[./stress]
  type = ComputeFiniteStrainElasticStress
[../]
(moose/modules/tensor_mechanics/test/tests/finite_strain_elastic/finite_strain_elastic_new_test.i)
