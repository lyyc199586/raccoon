NucleationMicroForce

This class computes the external driving force for nucleation given a Drucker-Prager strength envelope. The implementation follows Kumar et. al. (2020).

Example Input File Syntax

    C++ Type:double

    Options:

    Description:The compressive strength of the material beyond which the material fails.

  • deltadelta

    C++ Type:double

    Options:

    Description:delta

  • tensile_strengthThe tensile strength of the material beyond which the material fails.

    C++ Type:double

    Options:

    Description:The tensile strength of the material beyond which the material fails.
