Gap Conductance Constraint

Computes the residual and Jacobian contributions for the 'Lagrange Multiplier' implementation of the thermal contact problem. For more information, see the detailed description here: http://tinyurl.com/gmmhbe9

The GapConductanceConstraint class is used specify a heat flux across a gap equivalent to where is the gap conductance, is the gap distance, is the temperature on the primary side of the mortar interface, and is the temperature on the secondary side. Constraint enforcement is done using lagrange multipliers.
