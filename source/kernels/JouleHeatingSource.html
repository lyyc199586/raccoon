JouleHeatingSource

Calculates the heat source term corresponding to electrostatic Joule heating.

Overview

The strong form for JouleHeatingSource is defined as

where is the electrostatic potential and is the electrical conductivity of the material. can either be an ADMaterial or a traditional Material. Within the heat conduction module, this default could currently be provided by ElectricalConductivity for general problems or SemiconductorLinearConductivity for more specific problems related to semiconductors.

This class inherits from the HeatSource class via DerivativeMaterialInterface and JvarMapInterface.

Example Input File Syntax

An example of how to use JouleHeatingSource can be found in the heat conduction module test transient_jouleheating.i.

[./HeatSrc]
  type = JouleHeatingSource
  variable = T
  elec = elec
[../]
(moose/modules/heat_conduction/test/tests/joule_heating/transient_jouleheating.i)
