Example Input File Syntax

Time derivative terms are ubiquitous in any transient simulation. The kernel block for a transient diffusion problem that demonstrates the VectorTimeDerivative syntax is shown below:

[Kernels]
  [./diff]
    type = VectorDiffusion
    variable = u
  [../]
  [./time]
    type = VectorTimeDerivative
    variable = u
  [../]
[]
