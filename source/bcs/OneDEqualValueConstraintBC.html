OneDEqualValueConstraintBC

Computes the \f$ \int \lambda dg\f$ term from the mortar method (for two 1D domains only).

The constraint is imposed using a Lagrange multiplier, a scalar variable.

More information about the mortar method may be found in the Constraints documentation.

Example input syntax

In this example, two 1D lines are connected using the OneDEqualValueConstraintBC as a mortar constraint. The sign to impose on the variable value on the boundary nodes is specified with the vg parameter. The Lagrange multiplier, lm, is also subjected to a NodalEqualValueConstraint to achieve the continuity at the interface.

[BCs]
  [./left]
    type = DirichletBC
    variable = u
    boundary = '1'
    value = 1
  [../]

  [./right]
    type = DirichletBC
    variable = u
    boundary = '2'
    value = 3
  [../]

  [./evc1]
    type = OneDEqualValueConstraintBC
    variable = u
    boundary = '100'
    lambda = lm
    component = 0
    vg = 1
  [../]

  [./evc2]
    type = OneDEqualValueConstraintBC
    variable = u
    boundary = '101'
    lambda = lm
    component = 0
    vg = -1
  [../]
[]
(moose/test/tests/mortar/1d/1d.i)

    Description:Component of the Lagrange multiplier

    C++ Type:std::vector<VariableName>

    Options:

    Description:Lagrange multiplier

    C++ Type:NonlinearVariableName

    Options:

    Description:The name of the variable that this residual object operates on

  • vgVariation of the constraint g wrt this surface (+1 or -1). Note: g = value1 - value2 = 0

    C++ Type:double

    Options:

    Description:Variation of the constraint g wrt this surface (+1 or -1). Note: g = value1 - value2 = 0
