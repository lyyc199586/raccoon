InterfaceDiffusionFluxMatch

buildconstruction:Undocumented Class

The InterfaceDiffusionFluxMatch has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

Enforce flux continuity between two different variables across a subdomain boundary
